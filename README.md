# Korean Workplace Tutor

An interactive AI-powered Korean language learning tool for workplace translation practice.

## Features

- Dynamic sentence generation with AI
- Three difficulty levels (Beginner, Intermediate, Advanced)
- AI-powered feedback on translations
- Session summaries with detailed analytics
- Real-time Korean translation practice

## Getting Started

### Prerequisites

- Node.js (v14 or higher) - Download from https://nodejs.org
- npm (comes with Node.js)
- An Anthropic API key - Get one at https://console.anthropic.com

### Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/korean-tutor.git
cd korean-tutor
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser

5. Click the settings icon and enter your Anthropic API key

## Usage

1. Select your difficulty level (Beginner, Intermediate, Advanced)
2. Read the English sentence
3. Type your Korean translation
4. Click "Check" to get AI feedback
5. Review the detailed error analysis
6. Click "Next" to continue
7. After 10 translations, see your session summary

## Deployment to Vercel
